# Complete Dashboard Tool  
  
Generate end-to-end dashboards: from raw CSV to domain insights to sophisticated visualizations.  
  
## Usage  
```json  
{  
    "tool_name": "complete_dashboard_tool",  
    "tool_args": {  
        "csv_path": "path/to/data.csv",  
        "chart_type": "auto|bar|line|scatter|histogram",
    }  
}

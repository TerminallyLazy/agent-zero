# Dashboard Tool  
  
Use this tool to generate interactive dashboards and insights from CSV data files.  
  
## Usage  
```json  
{  
    "tool_name": "dashboard_tool",  
    "tool_args": {  
        "csv_path": "path/to/data.csv",  
        "chart_type": "auto|bar|line|scatter|histogram"  
    }  
}

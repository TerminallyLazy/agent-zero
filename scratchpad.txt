# Agent Zero MCP Server Integration - Task Notes

## Task: Add MCP Server at http://localhost:5000/mcp/sse ✅ COMPLETED

### Key Findings:
1. **MCP Configuration Location**: `tmp/settings.json` under the "mcp_servers" key
2. **Configuration Format**: JSON string containing an array of server configurations
3. **Server Types Supported**:
   - Local Stdio Servers (command/args based)
   - Remote SSE Servers (URL based) - This is what we need
4. **Handler**: `python/helpers/mcp_handler.py` manages all MCP operations
5. **Integration**: MCP tools are prefixed with server name (e.g., `server_name.tool_name`)

### Remote SSE Server Configuration Template:
```json
{
    "name": "server_name",
    "description": "Optional description",
    "url": "http://localhost:5000/mcp/sse",
    "headers": {}, // Optional HTTP headers
    "timeout": 5.0, // Optional connection timeout
    "sse_read_timeout": 300.0, // Optional read timeout
    "disabled": false
}
```
